1. How does javascript work?

Ans: JS is a single-threaded and single concurrent programming language, which means it can perform only one duty at a time., or in other words, one line of code. It is an interpreted programming language that, like most scripting languages, uses dynamic typing to provide type safety at runtime.

2. What does javascript event loop do?

Ans: JavaScript's runtime paradigm is event-driven, with the event loop in charge of executing the code, collecting and analyzing events, and executing queued sub-tasks.

3. What is the difference between local storage and session storage difference

Ans: SessionStorage is a type of storage that is comparable to LocalStorage. While data in LocalStorage does not expire, data in SessionStorage is deleted when the page session ends. When a document is loaded in a browser tab, a unique page session is formed and allocated to that tab.
